# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Instead, please email **security@opencongress.app** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work with you to address the issue before any public disclosure.

## Supported Versions

We only support the latest version on the `main` branch.

## Scope

This policy covers the OpenCongress web application and its API. It does not cover third-party services (Congress.gov API, Google OAuth, etc.).
